    Welcome!

    Continued by the CMRC 2017 (The First Evaluation Workshop on Chinese Machine Reading Comprehension，CMRC2017), in this year, the Second Evaluation Workshop on Chinese Machine Reading Comprehension (CMRC 2018) will be held in Changsha on October 20-21, 2018.

    In this year, we are focusing on the Span-Extraction Reading Comprension, which is similar to the SQuAD dataset.

    We will release the first Chinese Span-Extraction Machine Reading Comprehension Dataset. The participants are required to set up a system that reads context, question and extract the answer from the context (the answer is a continuous span in the context). In order to preserve the integrity, we will not release the test set to the participants. The participants are required to submit their system to the Codalab platform and evaluate on the development set, and the officials will replace the development set to the hidden test set to produce the final result.
